## Connecting to the Wireless Network/Wifi

* The **asu guest** wireless network is *available to all participants* but limited in capabilities.
* The **eduroam** network is available to anyone whose institution participates in [eduroam](https://eduroam.org/). If you need to set up anything, talk to your institution's IT support *before* coming to ASU.
* The **asu** network is only available to members of Arizona State University.

### "asu guest" network

Full instructions for connecting to the **asu guest** wireless network can be found in the [ASU FAQ: How do I connect to ASU Guest Wireless](https://asu.my.salesforce-sites.com/kb/articles/FAQ/How-Do-I-Connect-to-ASU-Guest-Wireless)

*You will need a means to read email or receive text messages because as part of the registration process, a code will be sent to you.*


In short: 
1. Connect to the SSID connection named **asu guest**.
2. Open a Web Browser and try and connect to the internet. If you have not recently connected to asu guest then a screen will be displayed to begin the ASU wireless network guest registration process.
3. From this screen, please select **Don't have an account?**.
4. Fill in the registration information and click **Register**. You can enter in a valid email address **OR** mobile phone number.

   Once the Register button is clicked then you will be shown a screen where you can initiate the sending of the login information. Depending on which option you filled in to receive the registration confirmation, the specific button for that option can be clicked to receive the registration information. Only the selected option will be available.
   
5. After receiving the username and password information, click **Sign On**.
6. On the **Guest Portal** sign in screen, fill in the **username** and **password** that you received via either text or email and select **Sign In**.
7. You will be presented with the ASU Acceptable Use Policy. Please read and review, by selecting **Accept** you agree and are completing the registration process to connect to ASU Guest Wireless.


*The "asu guest" network is bandwidth-limited. Make sure that you install any required packages before coming to ASU.*
